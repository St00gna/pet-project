# 💱 Currex --- сервіс відстеження курсів валют (НБУ)

Веб-сервіс для відстеження курсів валют за допомогою офіційного API
Національного банку України (НБУ).\
Проєкт автоматично завантажує курси для обраних валют за розкладом,
надає зручну адмін-панель для управління, дозволяє генерувати CSV-звіти
та має готове REST API.

------------------------------------------------------------------------

## 🛠 Технологічний стек

-   **Backend:** Python 3.12, Django 6.0, Django REST Framework\
-   **База даних:** PostgreSQL\
-   **Фонові задачі:** Celery + Celery Beat\
-   **Брокер повідомлень:** Redis\
-   **Інфраструктура:** Docker + docker \
-   **Документація API:** Swagger UI (drf-spectacular)

------------------------------------------------------------------------

## 🚀 Інструкція із запуску (Docker)

### 1. Клонування та збірка

``` bash
git clone <посилання_на_ваш_репозиторій>
cd currex
docker up -d --build
```

------------------------------------------------------------------------

### 2. Налаштування бази даних

``` bash
docker exec currency_web python manage.py migrate
docker exec currency_web python manage.py createsuperuser
```

------------------------------------------------------------------------

### 3. Первинне наповнення даними

``` bash
docker exec currency_web python manage.py load_currencies
```

АБО

-   Відкрити: http://localhost:8000/admin/
-   Розділ: **Currencys**
-   Натиснути **⬇️ Завантажити актуальні дані з НБУ**

------------------------------------------------------------------------

## 🎯 Як користуватися

### Адмін-панель

1. Налаштувати відстежування валют
-   Відкрити: http://localhost:8000/admin/
-   Розділ: **Currencys**
-   Позначити `is_tracked` для потрібних валют (USD, EUR, PLN)
-   Натиснути **Save**

2. Подивитися статистику
-   Відкрити: http://localhost:8000/admin/
-   Розділ: **Exchange rates**
-   Натиснути **Save**

------------------------------------------------------------------------

### Генерація CSV

Отримати курси зараз:

``` bash
docker exec currency_web python manage.py create_csv
```

За конкретну дату:

``` bash
docker exec currency_web python manage.py create_csv --date 2026-03-20
```

Файли зберігаються у папці:

    result/

## ⚙️ Автоматизація (Celery)

-   Задачі виконуються автоматично
-   За замовчуванням: щодня о **09:00** (для тестування стоїть кожну хвилину, о 9:00 закоментовано)

Перегляд логів:

``` bash
docker logs -f currency_celery_worker
```
